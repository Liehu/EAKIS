#### 任务编排（演练计划）— 后端无API

- **简单模式（扫描任务）**：选模板+选目标 → 直接执行单个扫描任务

```typescript
// api/orchestrations.ts 需新建

// POST   /v1/orchestrations/{id}/report         → 生成演练汇总报告
```

**后端实现参考：** 需新建 `Orchestration` ORM模型（演练计划）+ `OrchestrationTask` 关联表。子任务本质上是普通Task，通过关联表连接到演练计划。

**数据模型草案：**

```sql
-- 演练计划表
CREATE TABLE orchestrations (
    id          UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    name        VARCHAR(300) NOT NULL,          -- 演练名称
    description TEXT,
    status      VARCHAR(50) NOT NULL DEFAULT 'draft',  -- draft|running|paused|completed|cancelled
    created_by_user_id UUID REFERENCES users(id),
    org_id      UUID REFERENCES organizations(id),
    created_at  TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_at  TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

-- 演练-任务关联表
CREATE TABLE orchestration_tasks (
    orchestration_id UUID NOT NULL REFERENCES orchestrations(id) ON DELETE CASCADE,
    task_id          UUID NOT NULL REFERENCES tasks(id) ON DELETE CASCADE,
    sequence         INT NOT NULL DEFAULT 0,     -- 执行顺序
    PRIMARY KEY (orchestration_id, task_id)
);
```

**前端Stub内容：** 演练计划列表页（表格：名称、状态、子任务数、进度、创建时间）+ 演练详情页（子任务列表+各子任务进度+汇总统计卡片）。

```typescript
// 1. 请求拦截器：注入JWT Token
client.interceptors.request.use((config) => {
  const token = localStorage.getItem('access_token');
  if (token) {
    config.headers.Authorization = `Bearer ${token}`;
  }
  return config;
});

// 2. 响应拦截器：401自动跳转登录
client.interceptors.response.use(
  (response) => response,
  (error) => {
    if (error.response?.status === 401) {
      localStorage.removeItem('access_token');
      window.location.href = '/login';
    }
    // ...existing error logging...
    return Promise.reject(error);
  }
);
```
