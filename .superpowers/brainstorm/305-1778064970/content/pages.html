页面结构与路由设计

8 个页面，侧边栏导航切换，子路由展开详情

路由表

React Router 路由配置
/                       → Dashboard（总览仪表盘）
/keywords               → Keywords（关键词列表）
/keywords/:id           → 关键词详情/编辑
/assets                 → Assets（资产列表）
/assets/:id             → 资产详情（含关联接口和漏洞）
/interfaces             → Interfaces（接口列表）
/interfaces/:id         → 接口详情
/pentest                → Pentest（渗透测试状态）
/pentest/:id            → 渗透详情
/vulnerabilities        → Vulnerabilities（漏洞库）
/vulnerabilities/:id    → 漏洞详情（含证据和修复建议）
/reports                → Reports（报告列表）
/reports/:id            → 报告详情/下载
/settings               → Settings（系统设置）

页面内容概览

总览仪表盘 Dashboard
• 4个指标卡片（资产数/接口数/漏洞数/进度）
• Agent 五层流程管线状态
• 高风险资产表格
• Agent 实时日志流
关键词生成 Keywords
• 关键词分类表格（业务/技术/主体）
• 权重和置信度展示
• 手动添加/删除关键词
• 关键词覆盖率统计
资产关联 Assets
• 资产列表（筛选：风险/类型/WAF/ICP）
• 资产详情（技术栈/端口/证书/漏洞）
• 风险等级标记和确认
• CSV/XLSX 导出
接口爬取 Interfaces
• 接口列表（筛选：类型/方法/优先级）
• 接口详情（参数/权限敏感度）
• 爬取方法标识（动态/静态）
• 测试优先级标记
自动渗透 Pentest
• 渗透进度和状态
• 实时测试日志
• 发现的漏洞列表
• 测试配置和参数
漏洞库 Vulnerabilities
• 漏洞列表（筛选：严重性/类型/确认状态）
• 漏洞详情（证据/请求响应/修复建议）
• CVSS 评分和 LLM 置信度
• 状态更新（确认/误报/已修复）
报告中心 Reports
• 报告列表和生成触发
• 报告状态和进度
• PDF/MD 下载
• 质量评分展示
系统设置 Settings
• Agent 配置管理
• Webhook 通知设置
• 系统健康检查状态
• 全局指标概览
