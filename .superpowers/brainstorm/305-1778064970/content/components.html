共享组件与数据流

复用组件层 + API 请求流 + 实时事件架构

共享组件清单

AppLayout
侧边栏 + 顶栏 + 内容区三层布局
侧边栏可折叠，高亮当前路由
顶栏显示任务选择器和状态徽章
AgentPipeline
五层 Agent 流程管线可视化
每层显示：图标 + 名称 + 进度条 + 状态
状态：完成(绿) / 运行中(蓝) / 等待(灰)
MetricCard
指标卡片：标签 + 数值 + 变化量
支持 ↑/↓ 趋势标识
用于 Dashboard 顶部指标区
AgentLog
实时日志流组件
三种日志级别：✓ 成功 / → 信息 / ! 警告
自动滚动到底部，支持暂停
RiskTag
风险等级标签组件
critical/high/medium/low/info
颜色映射：红/橙/黄/绿/蓝
StatusBadge
状态徽章：运行中/空闲/完成/失败
支持 ● 动态脉冲动画（运行中状态）
用于顶栏和 Agent 管线
